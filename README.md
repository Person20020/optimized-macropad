# Optimized Macropad

This macropad is built to be in the shape of the optimized compact layout for 17 squares within a larger square. It has 17 keys with rgb backlights. It uses QMK Vial to allow the keys to be customized.

<img width="300" alt="Optimized Macropad" src="https://github.com/user-attachments/assets/31676650-c65e-4560-9eb6-0bad85f7f5c5" />

## PCB

There are two parts for the PCB. The main part holds the switches and the secondary small one allows the microcontroller to be attatched directly under in a more compact way.
